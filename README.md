SORTING VISUALIZER (Bubble Sort vs Insertion Sort)
--------------------------------------------------

DESCRIPTION
This Python script visualizes how Bubble Sort and Insertion Sort work side-by-side using bar charts.

REQUIREMENTS
You need Python and the matplotlib library. Install the library using this command:
pip install matplotlib

HOW TO RUN
1. Save the script as sorting.py
2. Open your terminal or command prompt.
3. Run the command:
   python sorting.py

SETTINGS
Open the script in a text editor to change these variables at the top:
- N: Number of bars to sort.
- SPEED: Animation speed (lower number is faster).
